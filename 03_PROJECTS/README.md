# 03_PROJECTS

Client and project-based organization directory.

## Purpose

This directory contains media organized by client projects and contracts. Each subdirectory represents a specific client engagement or project timeline.

## Structure

```
03_PROJECTS/
├── 2022_SageGrouse/
├── 2023_Wildlife_Contract/
└── [YEAR]_[ProjectName]/
```

## Usage

- Create subdirectories using format: `YYYY_ProjectName`
- Store finalized, client-deliverable media here
- Each project folder should contain its own organization structure
